# Flex (FlexBox)
Flexbox component for defining flex containers quickly with directional helper components baked in.

## Components
`<Flex/>` For general flex containers (use if you know directions will have to change for component).
`<Flex.Row/>` For row aligned content.
`<Flex.Col/>` For column aligned content.
`<Flex.RowReverse/>` For row reverse aligned content.
`<Flex.ColReverse/>` For column reverse aligned content.

## Props
`direction` props is omitted for all directional Components
